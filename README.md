# go-interpreter

「Go 言語でつくるインタプリタ」を読んでインタプリタを作ってみる。

## 1. 字句解析

```
go test ./monkey/lexer
```

<details>
<summary>メモ</summary>
字句解析器にソースコードを与えて初期化し、繰り返し NextToken()を呼ぶことでソースコードを読み進めていく。
トークンごとに、文字ごとに進んでいく。
</details>
